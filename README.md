## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

    NodeJS
    Npm

### Setup

1. git clone https://github.com/Kumilachew-g/Calculator-React.git
2. cd Calculator

### Install

    npm install

### Usage

    npm  start
